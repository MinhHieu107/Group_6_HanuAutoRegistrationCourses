# PRD: HANU Automatic Course Registration Website ## 1. Project Overview The HANU Automatic Course Registration Website is a specialized web-based platform designed to streamline and automate the course enrollment process for students at Hanoi University. It addresses the common challenges of manual registration by allowing students to pre-configure their preferences and letting the system handle the heavy lifting during high-traffic registration windows. ## 2. Target Audience - **Students:** HANU students needing an efficient way to search, plan, and register for courses. - **Administrators:** Staff responsible for managing course data, schedules, registration rules, and user accounts. ## 3. Key Functional Pillars ### A. Student Portal - **Account Management:** Secure registration (Gmail) and authentication. - **Course Discovery:** Advanced search and filtering (by department, credits, schedule). - **Academic Planning:** Real-time timetable simulation with automatic conflict detection. - **Automated Enrollment:** Configuration of "Preferred Course Lists" with priority levels and automated submission logic. - **Tracking:** Real-time notifications and comprehensive registration history. ### B. Automated Processing (The "Engine") - **Queue Management:** Handling high-concurrency requests during registration openings. - **Constraint Validation:** Real-time checking of prerequisites, slot availability, and credit limits. - **Priority Logic:** Executing registrations based on user-defined priority levels. ### C. Administrative Suite - **Control Center:** Defining registration windows (start/end times) and global rules (credit caps). - **Curriculum Management:** CRUD operations for courses, sections, and schedules. - **User Governance:** Oversight of student accounts (locking/unlocking). ## 4. Non-Functional Requirements - **Performance:** Sub-3-second page loads; sub-2-second search results. - **High Availability:** 99% uptime, especially during peak windows. - **Security:** Role-based access control and data protection. - **Usability:** Intuitive, "no-training-required" interface consistent across desktop and mobile. ## 5. Success Metrics - Reduction in manual registration errors. - Decreased system downtime during peak registration hours. - High student satisfaction scores for the "Auto-Registration" feature.
